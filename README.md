# Fetch Module Project End

## ✨ Requirements

1. Node >= 18.x
2. Git Bash (Windows users)

## ✨ Usage

1. `npm install`
2. `npm run dev`

## ✨ Design

[mock site](https://bloominstituteoftechnology.github.io/W_S11_Fetch_Project)

## ✨ Endpoints

- `[GET] /api/dogs`
- `[GET] /api/dogs/breeds`
- `[DELETE] /api/dogs/:id`
- `[POST] /api/dogs`
- `[PUT] /api/dogs/:id`
